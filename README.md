# Port Scanner

A Python-based TCP port scanner built to explore **network reconnaissance**, **service enumeration**, and **socket programming** in **authorized lab environments**.

This project was created as part of my cybersecurity learning journey to better understand:
- how open ports expose services
- how attackers and defenders map network attack surfaces
- how Python can be used for low-level network interaction
